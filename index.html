Build & Chill Autoposter

A personal TikTok content scheduling tool

Build & Chill Autoposter is a personal productivity tool that uses the TikTok API to schedule and publish video content, read account analytics, and manage TikTok posts from a single dashboard. It is built and operated by a single developer for personal use.

What the App Does

  • Schedules and publishes videos to TikTok using the official TikTok API
  • Reads basic account information (username, avatar) to confirm account connection
  • Retrieves video performance analytics (views, likes, shares) for personal reporting
  • Stores all credentials locally — no data is sent to third-party servers

TikTok API Usage

This app uses the TikTok Content Posting API and TikTok for Developers platform. It requests the following permissions:

  • user.info.basic — display account name and avatar in the dashboard
  • video.list — retrieve published videos to show analytics
  • video.upload — upload and publish scheduled video content

Who Uses This App

This app is for personal use only by the developer. It is not a public product, SaaS tool, or commercial service. No third-party user data is collected or shared.
